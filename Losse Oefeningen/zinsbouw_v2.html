Nederlandse Zinsbouw

Leer hoe Nederlandse zinnen werken — stap voor stap

🔑 De Formule

ONDERWERP + WERKWOORD + REST + 2e WERKWOORD
(wie?) + (doet?) + (wat/wanneer/waar?) + (infinitief)

Hoe werkt het?


In het Nederlands staat het werkwoord ALTIJD op positie 2. Dit is de belangrijkste regel! Alles daarna is de 'rest': tijd, object, plaats. Als er een 2e werkwoord is (bij willen, moeten, kunnen, gaan), gaat dat werkwoord naar het EINDE van de zin.

PositieWat?VraagVoorbeeld
1OnderwerpWie? Wat?Ik, zij, de man, wij
2WerkwoordDoe-woordwerk, is, heeft, wil
3TijdWanneer?elke dag, morgen, nu
4ObjectWat? Wie?het pakket, de machine
5PlaatsWaar?op kantoor, in de fabriek
∞2e werkwoordEinde!werken, leren, versturen

📗 Voorbeelden A1 — Eén werkwoord

A1
Ik werk elke dag in de fabriek.Onderwerp + werkwoord + tijd + plaats
Zij spreekt drie talen op het werk.Onderwerp + werkwoord + object + plaats
Wij hebben pauze om 12 uur.Onderwerp + werkwoord + object + tijd
De machine is vandaag kapot.Onderwerp + werkwoord + tijd + rest
Hij stuurt het pakket naar de klant.Onderwerp + werkwoord + object + richting

📘 Voorbeelden A1+ — Twee werkwoorden

A1+
Bij willen, moeten, kunnen, mogen en gaan gaat het 2e werkwoord (infinitief) naar het EINDE van de zin.
Ik wil goed Nederlands leren.wil = werkwoord 1 (positie 2) · leren = werkwoord 2 (einde)
Zij moet elke dag op de machine werken.moet = ww1 · werken = ww2 achteraan
Wij kunnen morgen de bestelling versturen.kunnen = ww1 · versturen = ww2 achteraan
Hij gaat volgend jaar supervisor worden.gaat = ww1 · worden = ww2 achteraan
Je mag hier niet roken.mag = ww1 · roken = ww2 achteraan

⚠️ Veelgemaakte fouten

Deze fouten hoor ik vaak. Herken je ze?

1. Werkwoord niet op positie 2

Ik elke dag werk in de fabriek.→Ik werk elke dag in de fabriek.
Zij drie talen spreekt.→Zij spreekt drie talen.

2. Twee werkwoorden: het 2e staat niet achteraan

Ik wil leren goed Nederlands.→Ik wil goed Nederlands leren.
Hij moet werken elke dag vroeg.→Hij moet elke dag vroeg werken.

3. Duits en Nederlands mixen

Ik wahr früher in München.→Ik was vroeger in München.
Ik heb 2 kleines Kind.→Ik heb 2 kleine kinderen.
Ik finisf de universiteit.→Ik ben klaar met de universiteit.

4. Onvolledige zinnen

Ik sturen mensen van werk.→Ik stuur mensen naar het werk.
Ik ook werk op expeditie.→Ik werk ook op de expeditie.

5. Verleden tijd vergeten

Ik ben in Nederland al 4 jaar. (= verleden + nu)→Ik woon al 4 jaar in Nederland.
Laatste jaar ik finish universiteit.→Vorig jaar ben ik klaar met de universiteit.
Tip: Leer eerst de goede volgorde. De rest komt later. Onderwerp + Werkwoord + Rest = altijd goed!

🚫 Ontkenning: niet & geen

ONDERWERP + WERKWOORD + REST + NIET
'niet' staat achteraan (of voor het 2e werkwoord)

Wanneer NIET? Wanneer GEEN?

NIETGEEN
Bij werkwoorden, bijvoeglijke naamwoorden, plaatsen Bij zelfstandige naamwoorden (een → geen)
Ik werk niet.
Het is niet goed.
Ik ga niet naar huis.		 Ik heb geen tijd.
Er is geen probleem.
Ik heb geen auto.

Voorbeelden met NIET

Ik werk vandaag niet.niet → achteraan de zin
De machine werkt niet goed.niet → voor het bijvoeglijk naamwoord
Hij kan morgen niet komen.niet → voor het 2e werkwoord

Voorbeelden met GEEN

Ik heb geen pauze vandaag.een pauze → geen pauze
Er zijn geen handschoenen meer.handschoenen → geen handschoenen
Wij hebben geen problemen op het werk.problemen → geen problemen
